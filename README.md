- Unity Built-in Render Pipeline 으로 제작된 개인 프로젝트입니다.
- 모바일 혹은 PC에서 실행 가능하며 직장인들의 점심 메뉴를 3D 캐릭터들의 달리기 경주로 결정하는 게임입니다.
- 참가선수들의 이름을 결정하고 경기에 참여시킬 수 있으며, 각각 속도나 컨디션에 관련된 파라미터가 유기적으로 작동하도록 세팅하였습니다.
- 이름 타입으로 지정된 메뉴들이 얼마나 우승했는지 로컬로 저장합니다.

## 2. 개발환경
- UNITY 2020.3.18f1 / Built-in Render Pipeline / C#
- Visual Studio 2018
- Windows 10
- 2025.01 ~ 2026.04 (16개월)
- 개발 1명 (1인개발)

## 3. 사용기술
- 디자인 패턴 : Single Tone 패턴으로 매니저 관리
- Spine Animation : 스파인 4.1 버전 사용, UI애니메이션 구현에 사용하였습니다.
- TMP : 레거시 텍스트를 TMP로 변환하는 작업을 진행하였습니다.

## 4. 기능구현
- 리뉴얼
    작업자가 모두 퇴사한 관계로 유니티 버전을 6로 상향하고 그래픽, 아트스타일, 셰이더 등 리뉴얼 작업을 진행하였습니다.
- 경일고 신규 콘텐츠 작업
    경일고등학교 협력으로 라이브커머스 콘텐츠를 신규 추가하였으며 서버와 연동하였습니다.
